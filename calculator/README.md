1. Создание нового проекта

Откройте терминал.
Создайте новый проект React, выполнив соответствующую команду.
Перейдите в созданную директорию проекта.
Установите библиотеку Material UI, выполнив команды для установки:
npm i @mui/material @emotion/react @emotion/styled

2. Создание компонента калькулятора

В папке `src` создайте новую папку с именем `components`.
Внутри папки `components` создайте файл с именем `Calculator.js`.

3. Настройка компонентов Material UI

В файле `Calculator.js` импортируйте необходимые компоненты из библиотеки Material UI.
Создайте функциональный компонент `Calculator`.
Определите состояния для хранения значений вводимых чисел и результата вычислений с помощью хука `useState`.
Создайте функции-обработчики для изменения значений вводимых чисел.
Создайте функции для выполнения математических операций: сложение, вычитание, умножение и деление, и для обновления результата.

4. Разметка компонента

Оберните содержимое компонента в `Container` из Material UI. Задайте атрибут `maxWidth`, чтобы ограничить ширину контейнера.

Добавьте заголовок с помощью компонента `Typography`. Задайте атрибуты `variant` для стиля заголовка и `gutterBottom` для отступа снизу.

Добавьте два текстовых поля для ввода чисел с помощью компонента `TextField`. Настройте следующие атрибуты:

- `label`: текст метки.
- `variant`: стиль поля ввода.
- `margin`: отступы.
- `fullWidth`: растягивание на всю ширину контейнера.
- `value`: текущее значение поля ввода.
- `onChange`: функция-обработчик для изменения значения.
- `type`: тип вводимых данных.

Добавьте четыре кнопки для выполнения математических операций с помощью компонента `Button`. Настройте следующие атрибуты:

- `variant`: стиль кнопки.
- `color`: цвет кнопки.
- `onClick`: функция-обработчик для выполнения операции.
- `fullWidth`: растягивание на всю ширину контейнера.
